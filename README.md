# god's eye
An facial recognition app
A gui implemenation of face recognition app.
The app is created using Python's Kivy.
based on @ageitgey face recognition:
https://github.com/ageitgey/face_recognition

## Prerequisites
- Tensorflow == 1.12
- Dlib == 19.8.1
- sklearn == 0.20.2
- kivy == 1.10.1
- openface
- OpenCv == 4.0.0
- sqlite3
- numpy
